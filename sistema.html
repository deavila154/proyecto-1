SISTEMA DE RIEGO DE LEONARDO DA VINCI

Autores

  • Dayana Carolina Ovalle Ninco
  • Diego Andrés Osuna Barón
  • Jose Miguel Bermúdez
  • Felipe Chaparro

Resumen

El proyecto consiste en la construcción y prueba de un sistema de riego basado en los diseños de Leonardo da Vinci, utilizando principios hidráulicos como el tornillo de Arquímedes y sistemas de bombeo...

Abstract

The project consists of the construction and testing of an irrigation system based on Leonardo da Vinci’s designs, using hydraulic principles such as Archimedes’ screw and water wheels...

Conclusiones

  • El sistema funciona pero requiere mejoras técnicas (motorización, sellado, inclinación)
  • Es viable para cultivos pequeños en Bogotá si se optimiza
  • Futuras líneas de trabajo: sensores, control automático, mejores materiales
